# Task Tracker App

A basic task tracker app built using Vue.js as frontend and json-server as backend

## Setup

```
npm install
```

## Run JSON-Server backend

```
npm run backend
```

## Run VueJS development server

```
npm run serve
```

## Compile for production

```
npm run build
```
